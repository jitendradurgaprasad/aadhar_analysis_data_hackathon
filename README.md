# Aadhaar Enrolment Data Analysis (UIDAI Hackathon)

## Objective
Analyze UIDAI Aadhaar enrolment data to uncover demographic, temporal, and regional trends that can support policy planning and operational decision-making.

## Dataset
Official Aadhaar enrolment dataset downloaded from data.gov.in.  
The data is split across multiple CSV files and merged programmatically.
